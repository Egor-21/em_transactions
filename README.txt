Описание функционала

Проект представляет собой приложение для учета финансовых транзакций. Основной функционал включает в себя:

- Просмотр баланса: Возможность просматривать текущий баланс с учетом доходов и расходов.
  
- Добавление транзакции: При добавлении новых транзакций вводятся дата, категория(Доход/Расход), сумма и описание транзакций
  
- Поиск транзакции:  Функция поиска транзакции по одному из критерию: дата, сумма, категория
  
- Редактирование транзакции: Возможно редактирование уже существующей транзакции. Для этого нужно ввести индекс транзакции и внести новые данные
  
- Сохранение данных: Данные о транзакциях сохраняются в файле, изменения в файле происходят после завершения работы программы
  
- Интерфейс командной строки: Весь функционал доступен через командную строку
